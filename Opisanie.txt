
### Проект

Дополните класс `Tasks` следующими методами экземпляра:

- `importCsv` получает строку CSV, преобразует ее в массив задач и сохраняет в переменной экземпляра `this.tasks` (необходимо перезаписать `this.tasks` на новый массив)
- `getCount` возвращает количество задач
- `getFirst` возвращает первую задачу
- `getLast` возвращает последнюю задачу
- `getUnformattedTasks` возвращает строку всех задач в нижнем регистре и разделенных запятой и символом пробела
- при первичной отрисовке таблица должна показывать `"0"` и `"N/A"`  для соответствующих ячеек



 /** @param {string} csvString */
    importCsv(csvString) {
        // пополняем массив
        return this.tasks.push(csvString.split(",")) ?? "N/A";
    }

    getCount() {
        // выводит длину массива
        return this.tasks.length ?? "0";
    }

    getFirst() {
        // выводит первый элемент массива
        return this.tasks[0] ?? "N/A";
    }

    getLast() {
        // выводит последний элемент массива
        return this.tasks.at(-1) ?? "N/A";
    }

    getUnformattedTasks() {
        // выводит весь массив в видестроки в нижнем регистре
        return this.tasks.join("; ").toLowerCase();
    }
}
